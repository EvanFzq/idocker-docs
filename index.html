iDocker

    iDocker

    最好用的docker管理工具

    快速开始沟通反馈

    PC/移动端兼容

    该工具做了移动端和PC端适配，采用不同交互设计逻辑，体验更好.

    Compose

    Compose 是 Docker 官方的开源工具，负责实现对 Docker 容器的快速编排.

    文件管理

    支持对系统文件进行创建、查看、修改、删除、复制、重命名等操作.

    导航页

    自带导航页，可以方便的跳转到 docker 容器的服务地址，支持内外网模式.

    文档预览编辑

    支持对常见的文本配置文件进行预览修改，方便修改服务配置，支持语法高亮.

    事件通知

    支持对异常登录、容器停止、重启等事件进行邮件通知!
